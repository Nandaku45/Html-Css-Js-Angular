Html stands Hyper text markup language which used to create and display the web pages to the users in good structure and styles using elements,tags,hyper links,images and other media

Elements

Elements are used to define or represent the content in different sizes and different styles

Elements Classification

  1. Elemnts uses tags whic are classified into two types:
    • Paired tags (both open tag and closed tag)

      • Ex: header tags (<h1>content</h1>)

    • Non-paired tags(only open tag)

      • Ex: Img tags (<img>)

  2. Elements are classified into two types
    • Blocked Elements
      Blocked Elements are used to display the specific content in seperate line without others join with them

      • Ex: div elements, header elements

    • In-Line Elements
      In-line elements are used to display the content in same line with others join with them

      • Ex: strong elements, img elements
